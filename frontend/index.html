FAQ
คำถามที่พบบ่อย
ข้อมูลของฉันปลอดภัยไหม? +
ไฟล์ PDF ของคุณจะถูกลบออกจากเซิร์ฟเวอร์ทันทีหลังจากแปลงเสร็จ เราไม่เก็บข้อมูลทางการเงินใดๆ ไว้ การส่งข้อมูลทั้งหมดเข้ารหัสด้วย HTTPS/TLS
PDF แบบ scan ได้ไหม? +
PDF ที่เป็น text-based (ดาวน์โหลดตรงจากแอปธนาคาร) จะทำงานได้ดีที่สุด สำหรับ PDF ที่เป็นภาพสแกน ระบบจะใช้ OCR แต่ความแม่นยำอาจลดลงเล็กน้อย แนะนำให้ดาวน์โหลด statement ผ่านแอปหรือ internet banking โดยตรง
Excel ที่ได้มีอะไรบ้าง? +
ไฟล์ Excel จะมีคอลัมน์: วันที่, รายการ, รหัสอ้างอิง, จำนวนเงินเข้า, จำนวนเงินออก, ยอดคงเหลือ และหมวดหมู่ (สำหรับ Pro) มี sheet สรุปยอดรวมด้วย
ยกเลิกได้ไหมถ้าไม่พอใจ? +
ยกเลิกได้ตลอดเวลาไม่มีค่าปรับ สำหรับแผน Pro มีทดลองใช้ฟรี 7 วัน ถ้าไม่พอใจยกเลิกก่อนจะไม่เสียเงิน
มี API ไหม? +
มีสำหรับแผน Business ครับ รองรับ REST API สามารถ integrate เข้ากับระบบบัญชีของคุณได้โดยตรง ติดต่อทีมงานเพื่อขอ documentation
